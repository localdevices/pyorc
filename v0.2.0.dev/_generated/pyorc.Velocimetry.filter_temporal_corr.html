pyorc.Velocimetry.filter_temporal_corr

Velocimetry.filter_temporal_corr(v_x='v_x', v_y='v_y', corr='corr', tolerance=0.1)[source]

Masks values with a too low correlation.

Parameters

  • ds – xr.Dataset, containing velocity vectors as [time, y, x]

  • v_x – str, name of x-directional velocity

  • v_y – str, name of y-directional velocity

  • corr – str, name of correlation variable

  • tolerance – float (0-1), tolerance for correlation value. If correlation is lower than tolerance, it is masked

Returns

xr.Dataset, containing correlation filtered velocity vectors as [time, y, x]
